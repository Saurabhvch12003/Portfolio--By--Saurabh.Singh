## ✨ Overview

This is a **modern, animated developer portfolio** built using **Next.js, TypeScript, and Tailwind CSS**.

It is designed to:

- Showcase real-world projects
- Highlight backend + full-stack expertise
- Provide a clean, recruiter-friendly experience
- Deliver a visually engaging UI with smooth animations

---

## ⚡ Features

- 🌌 Animated particle background
- 🎬 Smooth UI animations using Framer Motion
- 👨‍🚀 Hero section with floating illustration
- 📱 Fully responsive design
- 🧠 Skills section aligned with real-world expertise
- 🗂️ Project showcase with images & tech stack
- 📩 Contact form with email integration
- 🔗 Social media buttons (GitHub, LinkedIn, LeetCode)
- 🔄 Page loader animation
- 📊 Scroll progress indicator
